# The Build-with-Tom AI Toolkit
*Every tool this page actually uses or has built with — what it's for, what it costs, and the honest one-liner. No filler.*

## Build something real
- **Lovable** — plain-English → working web app with payments. The fastest non-coder path from idea to sellable product. Free tier works. → https://lovable.dev
- **Claude (+ Claude Code)** — the writing/coding/reasoning engine behind this entire page; Claude Code runs my whole publishing pipeline. → https://claude.ai
- **Cursor** — AI code editor for when you outgrow app builders. → https://cursor.com
- **Replit / v0 / Bolt** — alternate app builders worth comparing on your use case.

## Create content
- **ChatGPT** — ideation, rewriting, images. → https://chatgpt.com
- **Midjourney** — best-in-class image generation. → https://midjourney.com
- **ElevenLabs** — realistic voiceover (my reels use free edge-tts; this is the upgrade). → https://elevenlabs.io
- **Suno** — full songs from a text prompt. → https://suno.com
- **Runway / Kling** — video generation for b-roll and effects.

## Automate the boring parts
- **n8n** — self-hostable workflow automation (the free Zapier). → https://n8n.io
- **Make / Zapier AI** — hosted automation if you don't want to self-host.
- **beehiiv** — newsletter platform with a real free tier. → https://beehiiv.com

## Research & stay current
- **Perplexity** — search that answers with sources. → https://perplexity.ai
- **NotebookLM** — feed it your docs, get summaries and audio overviews. → https://notebooklm.google

**My actual stack for this page:** Claude Code (pipeline) + free Meta Graph API (publishing) + edge-tts (voiceover) + Python/ffmpeg (rendering) — total software cost $0/mo. Comment SYSTEM on any post for the full architecture.

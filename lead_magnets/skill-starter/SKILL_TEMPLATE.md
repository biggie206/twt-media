# Claude Skill Starter Template
*From @build.withtom — the free template from the Claude Skills post.*

Copy this file, fill in the blanks, and you have a working Claude Skill.
A skill is just a text file that teaches Claude to do one task YOUR way,
every time — no code required.

---

```markdown
---
name: your-skill-name-in-kebab-case
description: One sentence saying WHEN to use this skill and WHAT it produces.
  Claude reads this line to decide when to trigger — make it specific.
---

# What this skill does
One short paragraph: the task, who it's for, what "done" looks like.

# Inputs I'll give you
- Input 1 (e.g. "a rough bullet list of what happened this week")
- Input 2 (e.g. "the audience: my clients / my boss / my newsletter")

# Steps — follow in order
1. First step. Be concrete ("Read the notes and group them into 3 themes").
2. Second step ("Write a 2-sentence summary per theme, plain language").
3. Third step ("Format as: subject line, greeting, themes, one next step").

# Rules (my way of doing it)
- Tone: (e.g. friendly but direct, no buzzwords, short sentences)
- Always include: (e.g. a clear call to action at the end)
- Never: (e.g. exceed 200 words, use exclamation marks, invent numbers)

# Example of a great output
(Paste ONE real example of the output done right. This is the single
biggest quality lever in the whole file.)
```

---

## The 10-minute quality checklist
1. **Name the trigger.** Does the `description` say exactly when to use it?
2. **One task per skill.** "Write my weekly client update" beats "help with email."
3. **Show, don't tell.** The example output teaches more than every rule combined.
4. **Test it.** Give the skill to Claude with real inputs. Fix what came out wrong.
5. **Version it.** When you improve a step, save a copy — buyers pay for maintained.

## Turning skills into a product (the $99 idea from the post)
- Bundle 3-5 related skills (e.g. "the freelancer pack": proposal, invoice
  follow-up, weekly client update, scope-creep reply, testimonial ask).
- Sell the BUNDLE, not the file: a Gumroad page with what each skill does,
  a demo screenshot, and who it's for.
- Price by outcome, not effort: what's an hour of the buyer's time worth?
